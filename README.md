## Introduction

1) SeekBar를 조정해서 최대 60분까지 분, 초 단위로 타이머를 설정할 수 있습니다.
2) 타이머가 진행되는 동안에는 틱톡 소리가 들리고, 시간이 종료되면 벨이 울리도록 했습니다.
3) 타이머를 종료하고 싶으면 0초로 설정하면 종료됩니다.
